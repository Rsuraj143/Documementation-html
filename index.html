Overview

Welcome to the documentation for the React js Consulting theme on this page, you will find all the steps you need to set up and use our theme. Please be sure to follow all of these steps and access the additional resources as needed.

Step 1 : Requirements

  • Node Js installation on your PC
  • VS Code editor

Step 2 : Download Code file from Server

  • After completing the purchase, download Code file to your computer. The zipped folder ends with “Consulting.zip” at the end of the filename.
  • Then After Extract the file(Right click to the selected file,then you can get the extract file option).

Step 3 : Installing Node jS

  • Download the Installer. Download the Windows Installer from Node JS
  • Official Website. https://nodejs.org/en
  • Install Node.jsand NPM. After choosing the path, double-click to install .msi binary files to initiate the installation process.
  • Check Node.js and NPM Version. (open a CMD and run a command "node -v")

Step 4 : VS Code Installation

  • Download the Visual Studio Code installer for Windows. install from official website https://code.visualstudio.com
  • Once it is downloaded, run the installer (VSCodeUserSetup-{version}.exe). This will only take a minute.

Step 5 : Open the Project folder in VS Code Editor

  • On the Visual Studio menu bar, choose File > Open > Folder, , and then browse to the code location
  • On the context (right-click) menu of a folder containing code, choose the Open in Visual Studio command. Choose the Open Folder link on the start window.

Step 6 : Open a New Terminal

  • Run a Command " npm install "
  • it will install all third-party libraries that we used in a project.
  • Run a Command " npm start "
  • it will redirect your system’s default browser and open your complete project.
